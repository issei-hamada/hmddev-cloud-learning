### 優先修正事項（全ロールに影響）
1. **コールドスタートの説明がない（R2-001）**: Lambdaの運用上の重要概念であり、CLF-C02でも出題される可能性がある。全ロールに影響するため追加すべき。
2. **CloudWatch Events / EventBridge の表記揺れ（R1-001）**: 現在のサービス名はEventBridgeであり、旧名称との関係を明示しないと混乱を招く。
3. **API Gateway の HTTP API 未言及（R2-005）**: REST APIとHTTP APIの2種類があることはCLF-C02の出題範囲内であり、言及が必要。
