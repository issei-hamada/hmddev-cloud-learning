AWS認定クラウドプラクティショナー（CLF-C02）向け学習コンテンツを作成するプロジェクト。

- 対象読者: IT初心者・クラウド未経験者
- コンセプト: 「なぜ必要か → 何が課題か → AWSでどう解決するか」の流れ
- 全17章（第0章〜第16章）+ 付録A〜C の構成
- kiro ベースで実装し、teacher / assistant / learning-reviewer エージェントを提供
- メンテナー向けに planner / writer / doc-reviewer エージェントを提供
- plan.md → draft.md → task.md → 各章執筆の流れで進行

**Why:** IT初学者が「基礎を理解した上での」合格を目指すため、単なる暗記ではなくアーキテクチャ理解を重視
**How to apply:** 各章の構成設計では、オンプレミス課題 → AWSサービスの流れを維持し、日常語での言い換えを常に含める
