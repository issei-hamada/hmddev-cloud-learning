# FPL レポート — MPL テンプレートへの追加セクション

FPL のレポートは `.claude/templates/mpl-report.md` をベースに、以下のセクションを追加する。

---

## 追加セクション1: ドメイン別スコア

総合スコアの直後に挿入する。

```
## ドメイン別スコア

| ドメイン | 配点 | 出題数 | 正答数 | 正答率 | スコアバー |
|---------|------|--------|--------|--------|-----------|
| 第1分野: クラウドのコンセプト | 24% | 7問 | <N>問 | <X>% | [<バー>] |
| 第2分野: セキュリティ | 30% | 9問 | <N>問 | <X>% | [<バー>] |
| 第3分野: テクノロジーとサービス | 34% | 10問 | <N>問 | <X>% | [<バー>] |
| 第4分野: 請求・料金・サポート | 12% | 4問 | <N>問 | <X>% | [<バー>] |
```

---

## 追加セクション2: 試験合格ライン分析

ドメイン別スコアの後に挿入する。

```
## 試験合格ライン分析

AWS Cloud Practitioner の合格ライン: **700 / 1000（70%）**

あなたの今回の結果: **<X>%**（<正答数> / 30問正解）

合格ラインを超えている場合:
  合格ラインを突破しています！本番試験に向けて自信を持って臨めます。

合格ラインを下回っている場合:
  合格ラインまで: あと **<N>問** 正解が必要です。
  > <弱いドメインへの具体的なコメント>
```

---

## 追加セクション3: 前回比較（2回目以降のみ）

レポートの末尾に追加する。

```
## 前回との比較（Attempt <前回> → Attempt <今回>）

| 項目 | 前回 | 今回 | 変化 |
|------|------|------|------|
| 総合 | <X>% | <X>% | <+/->X% |
| 第1分野 | <X>% | <X>% | <+/->X% |
| 第2分野 | <X>% | <X>% | <+/->X% |
| 第3分野 | <X>% | <X>% | <+/->X% |
| 第4分野 | <X>% | <X>% | <+/->X% |

> <改善・維持・低下のコメント>
```
